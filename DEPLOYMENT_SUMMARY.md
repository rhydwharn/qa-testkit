## 🆘 Troubleshooting

### Build takes longer than 10 minutes?
- This shouldn't happen, but if it does, check Vercel status
- You can manually restart: Dashboard → Deployments → Redeploy

### See "ERROR" status?
1. Click on deployment
2. Go to "Runtime logs"
3. Look for error messages
4. Common fix: Redeploy from dashboard

### Can't connect to database?
- Run migrations after build completes:
  ```bash
  DATABASE_URL="..." pnpm run db:push
  ```

### App shows blank page?
- Hard refresh: Ctrl+Shift+R (or Cmd+Shift+R on Mac)
- Check browser console (F12) for errors
